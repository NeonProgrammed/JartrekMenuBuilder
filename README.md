# Jartrek Menu Manager

Python Version: 3.9

Requires the following packages:

Flask,
Flask-Login,
Flask-SQLAlchemy,
Jinja2,
MarkupSafe,
SQLAlchemy,
Werkzeug,
click,
greenlet,
itsdangerous,
pip,
python-dotenv,
setuptools,
wheel


How to run:
-
- Install python3 (minimum 3.2)
- Run main.py using python.exe in venv/scripts
- Open ip-address from console in browser
